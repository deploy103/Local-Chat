# Local Chat

IP로 사용자를 구분하는 로컬 실시간 채팅 시스템입니다.

## 기능

- 4자리 랜덤 참여 코드로 방 생성과 참가
- Socket.IO 기반 실시간 메시지
- IP 기반 사용자 구분 및 방장 표시
- 방장 전용 방 터뜨리기: 메시지와 업로드 파일 삭제 후 방 종료
- 파일 업로드: 한 번에 최대 10개, 총 100MB, 24시간 후 자동 삭제
- 채팅 검색과 방별 검색어 기록
- 접속자 목록, 타이핑 상태, 초대 코드 복사, 드래그 앤 드롭 업로드
- 방, 메시지, 파일 메타데이터는 `data/chat.sqlite`에 저장
- 파일 본문은 메모리에 올리지 않고 `uploads/`에 저장

## 실행

Node.js `22.5` 이상이 필요합니다. 현재 개발 환경은 Node.js `24`에서 확인했습니다.

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000`으로 접속합니다.
같은 네트워크의 다른 기기는 서버 PC의 IP와 포트로 접속하면 됩니다.
